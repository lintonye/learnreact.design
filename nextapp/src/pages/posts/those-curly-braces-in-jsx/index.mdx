In JSX, `{ }` is a placeholder waiting to be filled. You can put in many things, a number, a string, an object or even a function. What you can put inside is an <span style={{color: 'white', background: '#9B51E0', padding: 5}}>Expression</span>.

Since it’s an expression, it can go pretty crazy. In fact, there’s no limit of what you can put inside. As long as it’s an expression, it’ll fit....

<img src={imgJsxComplexPng} alt={`jsx-complex`} width={1402} height={696} />

Sometimes you can even put a JSX tag inside the { }, because JSX
tags are <span style={{color: 'white', background: '#9B51E0', padding: 5}}>Expressions</span> too!
