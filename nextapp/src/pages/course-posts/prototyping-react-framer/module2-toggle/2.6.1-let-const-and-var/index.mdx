### What's so good about a constant?

You might ask whatâ€™s good about a constant if we cannot change its value. Well, just like many things in life, adding some limitations can bring benefits. This barrier can be exactly what we want. For example, if you are dealing with a physics game, you might need to add some physical constants such as gravity that we don't want to accidentally change.
